# AquamanSite
Movie Trailer framework using Atomic Design process (Winter 2019).
[Example framework](https://github.com/ewuweblab/movie-trailer-framework/blob/master/README.md) 

## Framework Dependencies
* [Normalize CSS](https://necolas.github.io/normalize.css/)
* [jQuery Libary](https://jquery.com/)

## Project Dependencies
* [Media Element Player](https://www.mediaelementjs.com/)
* [Layout Template](https://codepen.io/geoffgraham/pen/jaqWMJ)
    ### Plugins

    * [Animate on Scroll js (AOS)](https://unpkg.com/aos@2.3.0/dist/aos.js)
